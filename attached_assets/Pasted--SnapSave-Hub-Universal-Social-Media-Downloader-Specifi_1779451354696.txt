# 🚀 SnapSave Hub — Universal Social Media Downloader Specification

Build a fully production-ready, modern, ultra-powerful web application called **SnapSave Hub** that supports downloading media from ALL major social media, video, streaming, and content-sharing platforms.

The app must automatically detect the platform from the URL and provide reliable downloading for videos and audio.

---

# 🌍 UNIVERSAL PLATFORM SUPPORT (VERY IMPORTANT)

The app MUST support ALL social media and media-sharing platforms supported by yt-dlp or compatible extractors.

This includes, but is NOT limited to:

## 🎥 Video Platforms

* Vimeo

* Twitch

# 🧠 AUTO PLATFORM DETECTION

The system must:

* Automatically detect platform from pasted URL
* Show platform logo dynamically
* Load platform-specific UI automatically
* Route to correct extractor automatically
* Handle unknown URLs safely
